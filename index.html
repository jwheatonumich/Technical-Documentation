Introduction

SQL is used to communicate with a database. SQL statements are used to perform tasks such as update data on a database, or retrieve data from a database. Six standard SQL commands can be used to accomplish almost everything that one needs to do with a database: "Select", "Insert", "Update", "Delete", "Create", and "Drop".

Select

The "Select" command is used to query data from a table.

For example:

  • Select c1, c2 FROM t; will query data in columns "c1" and "c2" from a table named "t".

  • Select * FROM t; will query data in all columns from a table named "t".

  • Select c1, c2 FROM t WHERE condition; will query data in columns "c1" and "c2" from a table named "t" where a condition is true.

  • Select DISTINCT c1 from FROM t; will query distinct rows in column "c1" in all columns from a table named "t".

Insert

The "insert" command is used to add rows of data into a table.

INSERT INTO t(c1,c2,c3)
VALUES(1,2,3);

The above code will insert a row where columns "c1", "c2" and "c3" have the values 1, 2, and 3.

INSERT INTO t(c1,c2,c3)
VALUES(1,2,3),
VALUES(4,5,6);

The above code will insert two rows of data into columns "c1", "c2" and "c3".

Update

The "update" command will replace existing data with a new value.

UPDATE t
SET c1 = 100;

The above code will update all values of column "c1" to be 100.

UPDATE t
SET c1 = 100,
SET c2 = 200;

The above code will update all values of column "c1" to be 100 and all values in C2 to be 200.

UPDATE t
SET c1 = 100,
WHERE c2 < 100;

The above code will update values of column "c1" to be 100 for all isntances where "c1" is less than 100.

Delete

The "delete" command will remove data from a table.

DELETE FROM t;

The above code will delete all data from table "t".

DELETE FROM t
WHERE c1>999;

The above code will delete all rows from table "t" where column "c1" is greater than 999.

Create

The "create" command will create a new table.

CREATE TABLE t (
id INT PRIMARY KEY
name VCHAR NOT NULL price INT DEFAULT 0;

The above code will create a table with the 3 columns:

  • An integer row named "id" that is the table's primary key.
  • A character key named "name" that cannot be null.
  • An integer row named "price" that defaults to zero.
Drop

The "drop" command will delete a table.

DROP TABLE t;

The above code will delete the table "t" from the database.
